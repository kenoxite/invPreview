# Inventory Preview

### Info

Shows a real-time 3D model of your character in the inventory screen, with automatic stance changes based on equipped weapons.

You can check the already packed version by visiting this addon's [Steam Workshop](xxx) page.

### Features

- Real-time character model display using Picture-in-Picture (PiP)
- Auto-rotating camera to view equipment from all angles
- Dynamic stance changes based on equipped weapons (primary, handgun, launcher)
- Synchronizes with inventory actions (Take/Put)
- Compatible with vanilla inventory and Better Inventory mod

### Requirements

- Picture-in-Picture enabled in Video Options (Low or higher)
